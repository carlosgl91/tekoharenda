# Bienvenido a Tekoharenda

**Tekoharenda** es el catálogo nacional de código abierto para datos espaciales e imágenes de Google Earth Engine (GEE) en Paraguay.

## Sobre este catálogo

El objetivo de este sitio es centralizar, documentar y facilitar el acceso a la información geoespacial del país. Aquí encontrarás:

* Descripciones detalladas de los conjuntos de datos.
* Fragmentos de código listos para usar en Google Earth Engine.
* Información técnica sobre bandas y resoluciones.

---

*Para sugerir un nuevo conjunto de datos o reportar un error, por favor abre un "issue" en nuestro repositorio de GitHub.*
