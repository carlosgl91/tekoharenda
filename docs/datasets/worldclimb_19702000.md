# WorldClimb 1970-2000

**Tipo de Dato:** climáticos
**Descripción:** Datos de WorldClimb 1970-2000:  temperatura mínima, media y máxima, precipitación, radiación solar, velocidad del viento, presión de vapor de agua y precipitación total. Además se incluyen variables bioclimáticas que se derivan de los valores mensuales de temperatura y precipitación para generar variables con mayor relevancia biológica. En total 26 raster de variables climáticas y derivadas

**Resolución Espacial:** 900 metros
**Fuente Original:** WorldClim
**Fecha de Publicación:** 1970-2000
